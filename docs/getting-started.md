# Getting Started with CopyDetect

CopyDetect is a 100% free plagiarism checker that analyzes your content using Google's Custom Search API (Google Search technology) and shows line-by-line matches with sources.

Official website: https://copydetect.in

---

## What You Can Check

You can submit content to CopyDetect using:
- Text input
- File upload
- Google Drive import
- Dropbox import
- Document links / URLs

CopyDetect will return:
- Line-by-line plagiarism results
- Source website links for matched lines
- A downloadable PDF report (where available)

---

## Typical Workflow

1. Open CopyDetect: https://copydetect.in  
2. Choose your input method (Text / File / Drive / Dropbox / Link)  
3. Submit your content for checking  
4. Review line-by-line results and sources  
5. Download the plagiarism report as PDF if needed  
6. (Optional) Create an account for saved history and premium API access
