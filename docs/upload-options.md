# Upload Options in CopyDetect

CopyDetect supports multiple ways to submit content.

## 1) Text Input
Use this option when you want to paste content directly.
Steps:
1. Choose "Text" input
2. Paste your content
3. Run the plagiarism check
4. Review results and sources

## 2) File Upload
Use this option when you have a document file.
Steps:
1. Choose "File Upload"
2. Select your document
3. Upload and run check
4. Review results and sources
5. Download PDF report if needed

## 3) Google Drive Import
Steps:
1. Choose "Google Drive"
2. Connect/select your Drive file
3. Submit for checking
4. Review results and sources

## 4) Dropbox Import
Steps:
1. Choose "Dropbox"
2. Connect/select your Dropbox file
3. Submit for checking
4. Review results and sources

## 5) Document Link / URL
Steps:
1. Choose "Link / URL"
2. Paste the public document link
3. Submit for checking
4. Review results and sources
