# CopyDetect Features

CopyDetect is a free plagiarism checker focused on clarity and source transparency.

## Core Features
- Free plagiarism checking
- Line-by-line match detection
- Source identification with clickable URLs
- Support for multiple input methods:
  - Text input
  - File upload
  - Google Drive
  - Dropbox
  - Document links / URLs
- PDF report download option

## Accounts & Premium
- Registered users can manage checks and reports
- Premium members can access CopyDetect API endpoints (for integrations)
