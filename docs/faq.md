# FAQ

## Is CopyDetect free?
Yes. CopyDetect is a free plagiarism checker.

## Does CopyDetect detect AI-written content?
No. CopyDetect is not an AI content detector. It is focused on plagiarism and source matching.

## How does CopyDetect find plagiarism sources?
CopyDetect uses Google's Custom Search API (Google Search technology) to check lines against publicly available web sources.

## Can I upload documents?
Yes. You can submit content via file upload, Google Drive, Dropbox, or links/URLs.

## Can I download results?
Yes. CopyDetect supports downloading plagiarism reports in PDF format.

## Do you offer an API?
Yes. Registered premium members can access CopyDetect API endpoints.
